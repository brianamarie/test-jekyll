    after_success and after_failure TL;DR

    These scripts are run after the main script, and depend on the outcome. Optional.

    What are after_success and after_failure?

    These scripts will run based on if the main scripts pass or fail. If the scripts within after_success or after_failure fail, the build is unaffected.

    For more information on after_success and after_failure, see the detailed documentation.
