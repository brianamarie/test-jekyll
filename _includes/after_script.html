• after_script
    after_script TL;DR

    Script that runs after the rest of the script process is complete.

    What is after_script?

    Just like the other script steps, this will run scripts. The difference is that these scripts will wait to run until the other steps are complete, and if scripts in after_script fail, your build will not be affected.

    When is after_script necessary?

    You would use the after_script block if there was code you wanted to run after every build regardless of success or failure, and you didn't want its failure to affect the build.

    after_script examples
    
        after_script:
          - build_something_important
