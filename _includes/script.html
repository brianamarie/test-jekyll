    script TL;DR

    Here, Travis runs the test script. By default, Travis runs the default tests for the set language. Optional.

    What is script?

    By default, Travis will just run the associated default tests for the language. In a node app, it would run npm test.

    You can change this behavior by adding a script step.

    When is script necessary?

    Use script when you want scripts to be run that aren't necessarily default to the language, but you want Travis to fail if they exit with a non-zero code.

    script examples
    
        script:
          - bundle exec rake build
          - bundle exec rake builddoc
